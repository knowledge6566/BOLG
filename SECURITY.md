# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest | ✅ Yes |
| Older versions | ❌ No |

## Reporting a Vulnerability

If you discover a security issue in this project, please report it by opening a GitHub Issue or contacting the repository owner through GitHub.

Please include:
- A description of the issue.
- Steps to reproduce it.
- Screenshots (if applicable).

Thank you for helping improve the security of this project.
